# Plant Clasifier CNN

> Projekt zaliczeniowy

Aplikacja webowa do rozpoznawania gatunków roślin oraz wykrywania chorób liści
na podstawie zdjęcia, zbudowana w oparciu o konwolucyjną sieć neuronową (CNN).

---

## Funkcje

- klasyfikacja gatunku rośliny na podstawie zdjęcia liścia
- Wykrywanie chorób liści (narazie tylko dla jabłoni)
- Interfejs webowy — wystarczy wgrać zdjęcie aby otrzymać wynik

---

## Obsługiwane rośliny

Apple, Blueberry, Cherry, Grape, Pepper, Raspberry, Soybean, Strawberry, Tomato

## Technologie

- **Deep Learning:** TensorFlow/Keras (CNN)
- **Backend:** Python
- **Frontend:** Streamlit
